**앨범이름 변경**
----
  앨범 업데이트.
  
  본인 앨범의 제목 등을 수정할 수 있습니다. jacket URL 을 수정하는 것도 가능!

* **URL**

  /album/<int:user_id>

* **Method:**

  | `PUT` |
  
*  **URL Params**

   **Required:**
 
   N/A

   **Optional:**
 
   N/A

* **Data Params**

  ```json
      {
         "title": "kimchi",
         "jacket": "http://www.naver.com"
      }
  ``` 

* **Success Response:**
  
  * **Code:** 200 <br />
    **Content:** 
    ```json
        {
            "okay": true,
            "album": {
                "id": 3,
                "user_id": 1,
                "title": "kimchi",
                "jacket": "http://www.naver.com",
                "cnt": 2,
                "public": true,
                "clip_ids": []
            }
        }
    ```
 
* **Error Response:**
    
  * **Code:** 406 NOT ACCEPTABLEHTTP_404_NOT_FOUND <br />
    **Content:**
    ```json
      {
          "okay": false,
          "msg": "not yours"
      }
    ```
    **Reason:** 자신의 앨범이 아닌 앨범을 조작하려고 할 때
